# Prisha Birthday Website

A small birthday website made for Prisha, with a countdown to 15 May, animated wishes, and a memory gallery using the uploaded `d1.jpeg`, `d2.jpeg`, and `d3.jpeg` photos.

## Pages

- `index.html` - birthday countdown in days, hours, and seconds.
- `cause.html` - animated reasons and wishes for Prisha.
- `last.html` - final birthday memories gallery.

## Preview

Open `index.html` in a browser, or run a local server from this folder:

```bash
python -m http.server 8000
```

Then visit `http://127.0.0.1:8000/index.html`.
